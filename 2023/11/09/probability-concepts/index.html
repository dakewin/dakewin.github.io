概率中的一些概念

一、概率、期望值、方差

  1. Random variable随机变量：风险资产的收益率是一个随机变量
  2. Outcomes结果：一个量，可能出现的结果
  3. Event事件：一组具体的实验结果
  4. Relationship among events
    1. Mutually exclusive events互斥事件: The events competing directly with each other that they would not happen at the same time.
    2. Exhaustive events 穷举事件：Events cover all possible outcomes.
    3. Independent events: A不依赖与B
    4. Dependent events: A的发生依赖与B
  5. 概率的定义：
    1. 任何事件E的概率是一个介于0-1之间的实数，即：0<= P(E) <=1
    2. 任何一个两两互斥切穷举的事件之和的概率为1
  6. 概率的类型
    1. Emprical probability 经验概率：根据历史数据所获得的相对发生频数作为事件发生概率的估计值
    2. Priori probability 先验概率：对于问题的逻辑推断所推导出来的概率
    3. Subjective probability 主观概率：一种基于个人或者主观判断而获得的概率
  7. Odds 胜率
    1. Odds for event E：P(E) / [1-P(E)], given a probability P(E)
    2. Odds against event E: [1-P(E)]/P(E)
  8. 无条件概率和条件概率
    1. Unconditional probability: P(A)
    2. Conditional probability: P(A|B)
  9. 联合概率和概率的乘法原则 Joint probability and multiplication rule
    1. Joint probability 联合概率：事件A和事件B同时发生的概率，被称作P(AB)
    2. Multiplication rule for joint probability 概率乘法法则：
      1. P(AB) = P（A|B）P（B）
      2. For independent rules：
        1. P(A|B) = P(A)
        2. P(AB) = P(A)P(B)
  10. 概率的加法原则 addition rule for probabilities
    1. P(A+B) = P(A) + P(B) - P(AB)
    2. 如果事件A和事件B是mutually exclusive，那么P(A+B) = P(A) + P(B)
